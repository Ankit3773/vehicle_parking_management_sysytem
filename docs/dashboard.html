Recommended Demonstration Flow

  1. Login with the administrator credentials.
  2. Open the dashboard and explain the live statistics.
  3. Add a vehicle entry using the recommended demo record.
  4. Show the allocated parking slot in the confirmation panel.
  5. Open the slot status page and highlight the newly occupied slot.
  6. Process vehicle exit from the billing module.
  7. Explain the generated bill and parking duration.
  8. Search the same vehicle number to show stored transaction history.
  9. Open the report page and explain entries, exits, and revenue.
